Anomaly wrote:
I created a batch file quite a while ago. Maybe some will enjoy using it. It automatically creates a backup of your auto save. Just ignore if you don't need it.

This just means you don't have to save manually. You'll have a save game file of each turn unless you choose another setting and set a certain interval. (even for Legendary mode where you can't save manually).


 Spoiler
The idea is that every new turn the auto save game file is backed up in a folder called after the number of the turn. I wanted to cheat a little on Legendary mode ^^

Unfortunately there's quite a user manual for it if you haven't used it before.

First off 4 conditions:
- The batch file needs to be placed inside "%Appdata%\The Creative Assembly\Rome2\save_games" or whatever folder is used as location to save your Rome2 save games.
- The same goes for GetTurnTool.exe (credits to RoninX2807) - it needs to be along side the batch file, in the same folder.
- What you also need in this folder is a shortcut to your Rome2 executable.
- Make sure the shortcut has the same name as the executable (shortcut shouldn't have the ".exe", though.) So, EXE:"Rome2.exe" and Shortcut:"Rome2".

Regarding the settings inside the batch file:

Set FolderName=CustomSaveFolder
(This is to be the name of the folder your saves will be saved in.)
Set Save=
(Best to leave empty, unless you know the exact name of your save game file.)
Set RemoveSaveInterval=5
(This is a funny feature, after you close Rome2 this feature can delete certain saves. Say you set this value to 5 and you start a new game. After you get to turn 54 you close Rome2, you'll only have the save game files for Turn 1,5,10,15,20,25,30,35,40,45,50 and 54) Set this value to "1" and no saves will be removed.)
Set GameEXE=Rome2
(Best to leave this as it is.)

I've only tested this batch for Rome2. Needless to say: you should start the game with the batch file. The batch should close shortly after you closed the game.
