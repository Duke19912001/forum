Ok i think i find the solutions for the all problems. For me this worked... idk for all of you. But u can try.
HOST (person who create game) -->
Install Reloaded + Crack (from installation folder) + Reloaded Update 1.01
Host need to start Tunngle and then start the game
And go to local to create the game
CLIENT (person who joins the game) -->
Install Reloaded + Crack (from installation folder) + Reloaded Update 1.01 + Steam-FIX
Client need to start Tunngle and Steam and then start the game
And go to local and join the friend

Why both can't have same fix >>>
If both use Relaoded u can play and save. But u can't load the game. 
If both use Steam-FIX u can play for some time (or not even) and get VAC error.

For me this worked. And others reported that this worked.
