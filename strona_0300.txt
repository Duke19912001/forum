[Data: Saturday, 14 Sep 2019, 18:14]
Idk if it was posted already but this is how to get rid of the damned intro videos

Here's a lil guide how to get rid of those pesky intro vids:

(do at your own risk)

download the brilliant modding tool called Pack File Manager (PFM) from here:
https://sourceforge.net/projects/packfi ... s/Release/

extract it somewhere - I recommend not to use the same folder in which you have installed Rome Total War 2 (RTW2)

when you start PFM for the first time, it asks for directories. When it asks for RTW2, choose its install path. Cancel the search queries for the other games (Shogun & Napoleon) if you don't wanna use PFM with them

If something went wrong here and PFM crashes with exceptions, delete gamedirs.txt in the PFM Folder and start over.

now, this is important: BEFORE you start PFM, open the Windows Explorer and go to RTW2\Data Folder and create a backup of the file movies.pack. (just in case something goes wrong)

start PFM and go to options > uncheck CA Packs Are Read Only

a prompt will appear > Click: I'm not afraid, coward!

go to > File > Open > and choose the RTW2\Data - Folder and open the file movies.pack with it

delete Intro.bik, intel_logo_english.bik and sega_logo_sting_hd.bik - alternativly you can replace them with a blank bik file. (Menu Bar > Files > Replace File)

Voilá, no annoying intros anymore \o/

found on https://www.reddit.com/r/totalwar/comme ... in_rome_2/
(link modified with latest PFM)
