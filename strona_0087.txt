[Data: Saturday, 26 Jul 2014, 12:29]
Shadowduke patch fix worked a treat, thanks, much appreciation.

Apex86
Remove all mods.
Try using a lower shader model to start, big frame hit for small graphical improvement.
Go to your advanced graphics settings, turn off all the effects, set trees grass shadows water and sky to low, set a lower shader model, start with 3, use catalyst control or ati tray tools to set AA and AF turn them off in game always set mine with ati tray tools much better performance than when i let any game control it, just use triple buffering in game graphics menu.

Should now run smooth as silk, if so, start turning the graphics back up, incremental, a bit at a time and check to see.
Process of elimination, you should be able to work through it slowly and find where you are getting the biggest hit graphically, although you should get pretty good performance with that system.

Grab the latest Ati beta`s made a small positive difference for me.

I use an old gpu ati5670 and the effects make the biggest difference to me

Check your background tasks, use a site like blackvipers to turn off unused services.

Just a few thoughts, hope this helps.
