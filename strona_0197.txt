Regarding localizations for patch 17:

I did a some testing; installed Total.War.ROME.II.Emperor.Edition.MULTi9-PROPHET and then compared german language with legit german Rome 2. I tested and compared pretty much every aspect of the game to see whats in german and if there is anything missing. So far both versions are the same.

So then i updated the game using Total.War.ROME.II.Update.v2.2.0.15666-v2.2.0.16019 (patch 17), and copy pasted the german localization for patch 17. Now when i start the game i get missing text and missing sounds. Then i deleted the old files (local_ge_rome2.pack and music_ge_rome2.pack), which are obsolete and arent present in legit Rome 2 anymore. (I also deleted local_en.pack, local_en_shared_rome2.pack, music_en_shared_rome2.pack, created during the update, since im not using english language).

Tried running the game again and now everything works as it should.

So to summarize:
1.) Install Total.War.ROME.II.Emperor.Edition.MULTi9-PROPHET (any language) + Total.War.ROME.II.Update.v2.2.0.15666-v2.2.0.16019
2.) Select desired language pack from https://mega.nz/#F!F9wECKYB!48NjkPRTYE_7faZ8bQGdvw
3.) Extract it into your Rome 2 directory and overwrite
4.) IMPORTANT: Delete local_ge_rome2.pack, music_ge_rome2.pack, local_en.pack, local_en_shared_rome2.pack, music_en_shared_rome2.pack - ( "ge" represents your desired language, ("tr", "fr", etc..))
5.) Play

Alternative way (better):
1.) Install Total.War.ROME.II.Emperor.Edition.MULTi9-PROPHET (any language) + Total.War.ROME.II.Update.v2.2.0.15666-v2.2.0.16019
2.) Go to Rome 2/data and delete all language files. You will recognise then by having .._de.. or .._en.. or .._tr.., or .._fr.. etc., in their name
3.) Select desired language pack from https://mega.nz/#F!F9wECKYB!48NjkPRTYE_7faZ8bQGdvw
4.) Extract it into your Rome 2 directory and overwrite
5.) Play
