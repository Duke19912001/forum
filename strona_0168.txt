Alright here it is. It would be great if someone can actually tell me which part of this fixes it. Because it didn't work before, and now it works. 

It works with the 3DM Emperor Edition 2.2 version I linked above ^

Just copy the files into the game root folder and overwrite everything.

What I also did was:

- Install the following from the game folder, in that exact order.
1. vcredist_x86-90.exe
2. vcredist_x86-100-sp1.exe
3. vcredist_x86-80-sp1.exe

- Add the game folder to exclusion list of your antivirus. NOD32 tends to block stuff and quarantines cracks, as do many others.
- Go into Rome2.exe properties and if there is an "Unblock" option, unblock it. And in compatibility set it to run as administrator.
- Delete the scripts file in your appdata/Creative Assembly/scripts (I don't think its necessary, only if your game isn't running at all) itll reset all the settings, so you'll need to set them up again.
