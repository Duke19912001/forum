I had the same problem. This worked for me.

You need a hex editor, easiest to use is this one.
http://www.chmaas.handshake.de/delphi/f ... m#download

Then, you need to open the .pack archive with said program and press delete EIGHT times. You know it´s done when the first letter in the boxes to the right is a p.

Finally, save that .pack archive and move it to your data folder.

Download Mitch's mod manager but make sure it is 2.0 not 3.0 as 3.0 will send you to steam page.

The mod manager 2.0 should show you the mods if you did all of the above. But it will not load them yet.

For that you need to start total war rome 2 normally and tick the user.scripts option in the mods tab which is in options on the left.

It will tell you to restart and that's what you do but you can start the game through the mod manager now and everything should work now.

I suggest testing out with this UI mod to see if everything works.
