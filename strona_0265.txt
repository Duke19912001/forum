[Data: Monday, 16 Jul 2018, 23:13]
Download mod manager 1.3 (should work) or 1.4.
Or use script . Google to learn how to use script to load mods. (in game enable script and movie mods in Settings > mods)

Find
C:\Users\YourPC\AppData\Roaming\The Creative Assembly\Rome2\scripts
Create file with normal notepad called user.script.txt
In that you should be able to add mods
Correct syntax

in user.script.txt add lines to load mods (their names)


mod "_divide_et_impera_release_12_Part1.pack";
mod "_divide_et_impera_release_12_Part2.pack";
mod "mods_better_siege_weapons_equipments_&_walls_hq.pack";
mod "copy full mod name ";

Mod must be put in data folder of Rome 2

Steam mods - editing with notepad ++ is required
Open mod with notepad ++
Delete all letters before PFH4 .
It is now converted to script load. If you use normal notepad you will just break the mod

Load order - mod on top of the list wins
