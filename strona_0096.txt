How change Russian language in English?

When you run the setup.exe - the pop up window should appear allowing you to select 2 languages from the drop down list - "Русский" (Russian) and "English". So, just select the English language and continue with installation. It should install English version after this. If you still see Russian language in game, also try opening the SKIDROW.ini file in your install directory and ensure that the line "Language=english" is present there. If you see "Language=russian" in this file - change it to "Language=english".
