[Data: Friday, 06 Dec 2024, 10:23]
^
mods via the launcher unfortunately dont work. only works via user.script.txt...


1. start the game, go to option -> mods and activate both. close game

2. copy the mod (.pack file) into the "Total War Rome II\data" folder

3. go to %AppData%\The Creative Assembly\Rome2\scripts

4. open "user.script.txt" | if it doesn't exist create it

5. add the mod you want to activate


 example
mod "toonsrome2aio1.pack";
mod "toonsrome2aio2.pack";

6. save "user.script.txt" and start the game

There is also a mod manager but I never tried it.
